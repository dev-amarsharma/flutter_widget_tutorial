# HTML Lists

## Overview
Lists are used to group related items in a readable structure. HTML supports ordered and unordered lists.

## Unordered lists
Use an unordered list when item order is not important.

```html
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>
```

## Ordered lists
Use an ordered list when sequence matters.

```html
<ol>
  <li>Open the editor</li>
  <li>Write the markup</li>
  <li>Preview the page</li>
</ol>
```

## Description lists
HTML also supports term-description pairs.

```html
<dl>
  <dt>HTML</dt>
  <dd>Defines structure</dd>
</dl>
```

## Best practices
- Use lists for grouped content instead of plain paragraphs.
- Keep each list item focused.
- Choose ordered vs unordered based on meaning.
