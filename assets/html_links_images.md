# HTML Links and Images

## Overview
Links connect pages and resources. Images display visual content. Both are core building blocks of the web.

## Links

```html
<a href="https://developer.mozilla.org/">Visit MDN</a>
```

The `href` attribute defines the destination.

## Images

```html
<img src="logo.png" alt="App logo" />
```

- `src` points to the image file.
- `alt` provides a text description.

## Relative vs absolute paths
- Relative paths point to files in your project.
- Absolute URLs point to full web addresses.

## Best practices
- Use descriptive link text.
- Add meaningful `alt` text for content images.
- Avoid using images when text would be clearer.
