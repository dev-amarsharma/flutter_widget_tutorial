
# JavaScript Operators

## Overview
Operators perform arithmetic, comparisons, logical checks, and assignments.

## Why this topic matters
Operators let you calculate values, compare data, and control flow in expressions.

---

## Core concepts
- Arithmetic: `+`, `-`, `*`, `/`, `%`.
- Comparison: `===`, `!==`, `<`, `>`.
- Logical: `&&`, `||`, `!`.
- Ternary: `condition ? value1 : value2`.

---

## Example
```js
const a = 5
const b = 3
console.log(a + b)
console.log(a === b)
console.log(a > b && b > 0)
const result = a > b ? "a is larger" : "b is larger"
```

---

## Best practices
1. Keep code readable and consistent.
2. Use the right feature for the job.
3. Prefer modern JavaScript syntax where appropriate.
4. Test your code in the browser or console.

---

## Common mistakes to avoid
1. Ignoring edge cases.
2. Using outdated syntax without reason.
3. Writing overly complex code.
4. Forgetting to handle errors or invalid input.

---

## Quick practice
1. Compare two values with `===`.
2. Use a logical expression with `&&`.
3. Write a ternary expression.
4. Evaluate an arithmetic expression and log the result.
