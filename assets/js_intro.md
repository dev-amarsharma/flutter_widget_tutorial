# JavaScript Introduction

## Overview
JavaScript is the language of the web. It runs in browsers and enables dynamic behavior, interactivity, and application logic beyond static HTML and CSS.

## Why this topic matters
Understanding JavaScript fundamentals is the foundation for building modern websites and web applications. It prepares you for DOM manipulation, event handling, asynchronous programming, and more.

---

## Core concepts
- JavaScript runs in the browser and can also run on the server with Node.js.
- Code can be loaded inline with `<script>` or from an external `.js` file.
- JavaScript uses statements, expressions, variables, functions, and objects.
- `console.log()` is the most common tool for inspecting values during development.

---

## Example
```js
const greeting = "Hello, JavaScript!"
function showGreeting(message) {
  console.log(message)
}
showGreeting(greeting)
```

---

## Best practices
1. Keep your code simple and readable.
2. Use an external script file for larger code.
3. Test code in the browser console or developer tools.
4. Learn the difference between JavaScript syntax and HTML structure.

---

## Common mistakes to avoid
1. Mixing up HTML and JavaScript syntax.
2. Forgetting to include `<script>` tags or using the wrong path.
3. Using `var` for modern code instead of `const`/`let`.
4. Ignoring browser console errors.

---

## Quick practice
1. Log a message to the console.
2. Create a variable and print its value.
3. Update a value and print it again.
4. Move the code into an external `.js` file.
