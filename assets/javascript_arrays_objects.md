# JavaScript Arrays and Objects

## Overview
Arrays store ordered collections. Objects store named properties. Both are essential for modeling data in JavaScript.

## Arrays

```javascript
const topics = ["HTML", "CSS", "JavaScript"];
console.log(topics[0]);
```

Common array operations:
- `push()`
- `map()`
- `filter()`
- `find()`

## Objects

```javascript
const user = {
  name: "Asha",
  level: "beginner"
};

console.log(user.name);
```

## When to use which
- Use arrays for ordered groups of items
- Use objects for named data fields

## Best practices
- Keep object shapes clear
- Use descriptive property names
- Choose arrays when order matters
